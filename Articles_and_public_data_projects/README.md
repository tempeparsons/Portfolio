# My public data analysis projects

## How shared data and open-source resources can improve the lot of humanity and their environment?

Since changing careers from the lab bench to bioinformatics, this is a question that I've often come to ask myself.  
At the bottom of it lies three things: better data measurements, better analysis, better sharing. 
<br>  
My most sharable experitise is in mass spectrometry data analysis. For my first article in this section, I've used some public ovarian cancer proteomics data to showcase some of the ways I've learnt to deal with missing values, missing controls and unbalanced datasets. Take a look at the article **better_differential_abundance_analysis** in this section.
